# CRUD System

"A simple task manager to track employee habits/performances" This is a simple task manager designed to make Employers' lives much easier, by tracking the habits and performances of employees. This allows companies and organizations to minitor efficient workflow, and ensure tasks are being met are qouta's are being adhere'd to and that there are no causes for distractions or future disruptions standard operating procedures. The system will also enable Employers, to ensure employees are content and happy about their placements and positions. essentally monitoring employee satisfaction.

The CRUD system (Create, Read, Update, Delete) solves the fundamental problem of managing the lifecycle of persistent data in software applications. It provides a standardized, secure, and efficient framework for users to interact with database records—such as creating accounts, viewing content, editing profiles, or deleting items—without needing direct, complex database access.
